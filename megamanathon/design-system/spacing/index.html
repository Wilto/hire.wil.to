Tailwind-Provided Utility Classes

The following utility classes are provided by Tailwind CSS, which is incorporated into the build process for the sake of custom utility classes and processing design tokens. These utility classes will only be generated and included in the global stylesheet when used in markup.

While useful in some instances, we caution against relying on them as a default approach to general spacing for the sake of maintainability.
